# Guías del Salón

Tres guías escritas en español simple, pensadas para que cualquier persona del salón pueda usar la aplicación sin saber de computadoras. Imprimibles. Si encontrás algo que falta o que está mal explicado, anotalo y lo agregamos.

| Guía | Para quién | Qué cubre |
|---|---|---|
| **[GERENTE.md](GERENTE.md)** | Mamá / quien atiende la caja | Recibir clientes, cobrar, citas, salidas de inventario, cuadre del día, pagos a trabajadoras |
| **[TRABAJADORA.md](TRABAJADORA.md)** | Las estilistas | Mis citas, atender, marcar materiales, ver ganancias |
| **[INVENTARIO.md](INVENTARIO.md)** | Quien carga productos al sistema | Agregar productos, elegir tipo, plantillas, salidas, conteo físico |

## Cómo usar estas guías

1. **Imprimir las que necesites.** Cada guía termina con un "Recordatorio rápido" pensado para imprimir y pegar cerca del lugar de trabajo (la caja, atrás del espejo de la trabajadora).
2. **Leerlas despacio la primera vez.** No hace falta memorizar nada — están para consultar.
3. **Marcar lo que no entiendas.** Si algo no queda claro, escribirlo en una libreta. Las guías se actualizan con el tiempo.

## Reglas básicas que aplican a todas

- **Si no estás segura, preguntá.** Es mejor preguntar dos veces que romper algo.
- **El sistema perdona errores.** Casi todo se puede corregir. Las atenciones se pueden reabrir, los materiales se pueden editar, los precios se pueden ajustar.
- **Cada persona ve solo lo que le corresponde:**
  - La trabajadora ve sus servicios y ganancias.
  - La gerente ve todo el día.
  - La admin ve todo de todos los días.

## Cuándo actualizar estas guías

Cada vez que aprendamos algo nuevo del día a día — un atajo útil, un error común, una función que no se usa — lo agregamos a la guía correspondiente. Estas guías son **vivas**, no se quedan congeladas.
