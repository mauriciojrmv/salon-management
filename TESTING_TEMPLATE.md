# Testing Report Template

Copy a block below, fill it in, and paste it to me. One block per issue.

---

## Bug Report
```
PAGE: (ej: /sessions, /dashboard, /clients)
DEVICE: (ej: celular Samsung A12, iPhone 13, PC Chrome)
ROLE: (admin / manager / staff)
WHAT I DID: (pasos exactos, ej: "Abrí un trabajo, agregué servicio corte, toqué pagar")
WHAT I EXPECTED: (qué debería pasar)
WHAT HAPPENED: (qué pasó realmente - error, pantalla en blanco, número malo, etc.)
SCREENSHOT: (si es posible, pegar imagen o describir qué se ve)
REPEATABLE: (sí / no / a veces)
```

---

## User Complaint (UX/Usability)
```
WHO: (ej: estilista 55 años, dueña del salón, recepcionista)
PAGE: (dónde estaba)
COMPLAINT: (qué dijo exactamente o qué hizo mal - en sus palabras)
CONTEXT: (qué estaba intentando hacer, cuánto tiempo le tomó, se frustró?)
SEVERITY: (bloqueante = no pudo continuar / molesto = pudo pero se quejó / confuso = preguntó qué hacer)
```

---

## Feature Request
```
WHO ASKED: (quién y su rol)
WHAT THEY WANT: (en sus palabras)
WHY: (qué problema resuelve, qué hacen hoy en su lugar - papel, WhatsApp, etc.)
HOW OFTEN: (diario / semanal / raro)
PRIORITY GUESS: (must-have / nice-to-have / someday)
```

---

## Elderly/Non-Tech Observation
```
WHO: (edad aprox, rol, experiencia con tecnología)
WHAT HAPPENED: (qué intentó hacer, dónde se perdió, qué tocó por error)
BODY LANGUAGE: (se frustró? pidió ayuda? dejó de intentar? tuvo las manos mojadas/con guantes?)
WHAT WOULD HELP: (tu opinión - botón más grande? menos pasos? texto más claro?)
```
